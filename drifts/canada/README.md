# Canada Working holiday visa

## 1. 신청서 작성
 
 - [CIC](https://www.canada.ca/en/immigration-refugees-citizenship/services/application/account.html)

#### 합격자
10일안애 수락해야 한다.

## 2. 준비 서류

20일 안에 아래 사항을 제출해야 한다.

- 건강검진
- 범죄기록회보서
- IMM5707
	- 가족관계증명서 같은 것
- CV Resume
	- Career in Korea
- 여권사본(스캔)
- 여권사진(이미지 파일)

## 3. Bio Metrics

Bio metrics letter 메일을 받으면, 30일 안에 아래 사항을 수행해야 한다.

- 비자 센터 방문
	- 지문
	- 사진

## 4. 최종 합격

명시된 유효 날짜를 확인한다.
