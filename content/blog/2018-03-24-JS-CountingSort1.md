## 2. Process
1. 0~99까지의 개수를 파악할 배열을 만든다.
2. 0~99까지의 각 값을 배열의 인덱스로 사용하여 개수를 추가한다.
3. 결과를 출력한다.

## 3. Code
```tsx
function countingSort(arr) {
    let result = [];
    for(let i=0;i<100;i++) {
        result.push(0);
    }

    for(let val of arr) {
        result[val]++;
    }
    return result;
}

function main() {
    var n = parseInt(readLine());
    arr = readLine().split(' ');
    arr = arr.map(Number);
    var result = countingSort(arr);
    console.log(result.join(" "));
}
```
